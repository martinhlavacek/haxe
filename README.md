# haxe-default
Haxe default sample.
This sample includes haxe, nodejs and gulp

Install
-------
Install all nodejs packages

        npm install
        
Install msignal (this is Massive library [msignal](https://github.com/massiveinteractive/msignal))

        haxelib install msignal
        

Build app
---------

        haxe build.hxml
        
Run demo
--------

        gulp
